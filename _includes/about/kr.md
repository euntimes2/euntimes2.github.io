# 안녕하세요 — **박상현**입니다

저는 **서울대학교 컴퓨터공학부**에서 공부하고 있고,  
요즘은 **AI, 3D 비전, 로보틱스**를 중심으로 배우고 실험하고 있습니다.

처음엔 **원자핵공학**을 전공했어요.  
그때도 재밌긴 했지만, 점점 이런 생각이 들더라고요.

> “분석만 하는 것보다,  
> 직접 만들어 보고 움직이는 걸 보고 싶다.”

그래서 전과를 했고, 그 뒤로는  
코드 짜고, 디버깅하고, 논문 읽고, 막혔다가 풀리고…  
그런 날들의 반복 속에서 조금씩 배우고 있습니다.

---

## 🌌 요즘 특히 빠져 있는 것들

저는 제 성향이 딱 이 두 가지 사이에 있는 것 같아요.

- **정확하게 만들고 싶어 하는 엔지니어**
- **세상이 어떻게 보이는지 궁금해하는 사람**

요즘 관심 있는 주제들은:

- **3D 컴퓨터 비전**  
  (SfM, 포즈 추정, 포인트 클라우드, Gaussian Splatting, NeRF)
- **표현 학습** — 모델이 기하/동작/불확실성을 어떻게 담는지
- **로보틱스** — 인지에 기반한 행동과 현실적인 제약
- **도구와 시스템** — 그냥 “데모”가 아니라, 사람들이 실제로 쓰는 것

저는 단순히,

> “돌아간다.”

보다,

> “왜 이렇게 돌아가는지,  
> 그리고 다른 선택을 했다면 어떻게 됐을지”

생각하는 게 더 재미있습니다.

---

## 🛠️ 해 보면서 배운 것들

- **Structure-from-Motion** 일부 직접 구현  
  (포즈 추정, 번들 조정, 재투영 오류 잡기)
- SVM 실험, 커널 튜닝, 최적화 과정 분석
- **SIFT 매칭 + RANSAC 호모그래피** 구현
- 3D 모션 데이터로 **flow-matching 실험**
- 실제 작업에 쓰이는 자동화/시각화 스크립트 만들기
- **FastAPI + SQL + 클라우드** 기반 백엔드 설계

제 프로젝트들은 보통 이렇게 흘러가요:

> 엉성한 초안 → 더 나은 구조 → 테스트 추가 → 다시 정리

그리고 사실,  
**버그가 제일 좋은 선생님**이었던 적이 많습니다.
